Super Mario Broz

Como jogar:

  • Use as teclas de seta direita e esquerda para movimentar com Mario;
  • Use a tecla espaço para pular e matar o Goombas (inimigos);
  • Use a tecla seta para baixo quando estiver em cima dos tubos para trocar de tela
  • Tem 3 vidas, se perder todas as vidas, é gameover;
  • Se pegar o cogumelo, escondido na caixa de surpresas o Mario transforma em Super Mario;
  • Tem no maximo 5 minutos para passar de cada fase, se não passar de uma fase em até 5 minutos, gameo ver;

Para iniciar o jogo, pressione a tecla Espaço.
